# Progress
 Прототип блока Progress для использования в мобильных web-приложениях. Основное предназначение блока отображать процесс выполнения процессов и их прогресс выполнения. 

 ## Функциональность

 - **Normal состояние**: Отображение прогресса от 0% до 100%
- **Animated состояние**: Вращение дуги прогресса
- **Hidden состояние**: Скрытие блока
- **Адаптивность**: Поддерживает вертикальную и горизонтальную ориентацию
- **API**: Программное управление состоянием блока

Данный компонент был сознан без большого участия html, так как подразумевается универсальное использование в разных частях любого приложения. В ходе разработки была использована декомпозиция компонентов, чтобы при необходимости, например, исключить блок меню и использовать только компонент progressBar для отображения статуса загрузки.


## Использование

1. Откройте `index.html` в браузере или по ссылке (https://leolatri.github.io/Progress/)
2. Используйте элементы управления:
   - **Value**: Введите число от 0 до 100
   - **Animate**: Включите/выключите анимацию
   - **Hide**: Скрыть/показать блок
